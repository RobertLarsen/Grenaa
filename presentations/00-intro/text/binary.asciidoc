Talsystemer - Binær
-------------------

Det binære talsystem = base 2

Består af tallene [0-1]

[source,txt]
------------------------------------------------
10011 = 1 * 2^0 +
        1 * 2^1 +
        0 * 2^2 +
        0 * 2^3 +
        1 * 2^4 +
------------------------------------------------
